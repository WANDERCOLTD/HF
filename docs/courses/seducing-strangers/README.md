## Ethics — non-negotiable design property

Persuasion is **dual-use**: the same mechanisms work for ethical persuasion (amplifying what is true) and manipulation (manufacturing what is not). This course teaches **recognition of misuse alongside use**. The `pers.ethics_dual_use` LO appears in **Module 1**, and every subsequent module has an Edge Cases sub-section naming the manipulation pattern for that module's principle(s). Do not strip this out when adapting the bundle.

## How to ingest in `/x/wizard`

1. Open `/x/wizard` on hf-dev VM (or whichever environment).
2. Paste the **chat opener** below into the wizard chat.
3. Wait for the wizard to confirm subject + audience + goal.
4. When the wizard surfaces the Teaching Materials panel on the right, drop all five `*.md` files (everything except this README) into it.
5. Watch the Sources panel — every doc should classify by its `hf-document-type` (no `sourceWarnings`).
6. Continue the chat to confirm the proposal playback.
7. Once "Can launch: YES" + all preconditions are met, click "Ready to create my course".

### Chat opener (paste verbatim)

```
I want to build a 5-call demo course on persuasion and sales for general adult
learners who've never used AI tutoring before. Voice-only, about 7 minutes per
call. Frame it around Josh Weltman's "Seducing Strangers" with Cialdini's
"Influence" as the underlying mechanism layer. Goal is for them to walk away
able to spot the seven persuasion principles being used on them in real ads
and pitches, apply them honestly in their own work, and catch the manipulative
misuses — fake scarcity, manufactured social proof, manipulative reciprocity.
Socratic style, ethics threaded through every module, never lectured at.
```

## Post-create admin steps (set in `/x/courses/[id]/config`, not in the wizard)

These fields are **not** Wizard-settable. After the wizard creates the course:

1. **`Playbook.config.skillScoringEmaHalfLifeDays = 4`** — tightens the mastery-graph response so the dashboard shows visible movement in a 5-call demo (default is 14, which is too slow at this scale). Same nudge as the Big Five course.
2. **Module-specific Edge Cases instructions** — these live in the Course Reference Teaching Approach as text and project to `teaching_rule` assertions automatically; no per-module override needed unless you want to tighten further.

## Source provenance + copyright

All TEXTBOOK files are **HFF-authored summaries** that paraphrase published findings. No verbatim text from any source book is reproduced. Specific named examples (Petrified Forest sign, Hare Krishna airport, Tupperware parties, etc.) are paraphrased and attributed, never quoted.

The QUESTION_BANK items are HFF-authored Socratic probes **in the spirit of** the Cialdini / Weltman / Heath frameworks — not reproductions of book content.
